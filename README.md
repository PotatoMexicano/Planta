# Planta

Joguinho da planta desenvolvido em C# para a disciplina de desenvolvimento de sistemas (C#).

O programa apresenta ao usuário uma planta e um ecosistema ao usuário, onde o mesmo pode controlar o fornecimento de água e luz para a planta, possibilitando modificar a situação da planta.

O programa possui um pequeno tutorial de como as situações funcionam e um quiz para ser respondido.
